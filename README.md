# コードテキスト
　本テキストはコードに関する入門的な知識をまとめたものです。音楽の基礎的な構築から初めて用語と基本知識の用意をし、各種コードを説明したあと、それらを組み合わせて進行を構成します。最終的にコード進行を使う、作ることができるようになることを目標としています。理論的な話も出てきますが、不要だと思った場合は読み飛ばしても大丈夫です。
